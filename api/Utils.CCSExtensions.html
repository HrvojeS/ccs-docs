    Enumerate<T>(IList<T>, Int32, Int32)

    Enumerates the list from (zero based) startIndex to endIndex (inclusive). If the startIndex is larger than endIndex, the enumeration is done backwards.

    Ex. startIndex is 2, endIndex is 4: the method yields members of the original list located at indices 2, 3, 4.
