    Class LandscapeCheckers

    Hold checker functions for landscape. Every method has public and private variant. Public methods are used to define types of arguments and can be used in code at will. Private functions are not to be used in code and are used only to be wrapped in other methods. Conditions in json config call private methods defined here with signature defined in it's public method.
